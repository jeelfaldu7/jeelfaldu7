### 📈 U.S. Consumer Behavior Under Inflation
`🥇 Code Jam Winner` `DS/ML Code Jam Feb 2026` `Macroeconomics` `Time Series`

Investigated whether post-COVID inflation permanently altered U.S. consumer behavior using FRED macroeconomic data. Trained an OLS regression model on pre-COVID data to forecast post-COVID behavior, revealing a 6x spike in prediction error confirming a permanent structural break.
- Applied multivariate ruptures changepoint detection identifying two distinct breaks in Jan 2020 and Dec 2022
- Concluded inflation fundamentally changed how Americans spend, save, and borrow
